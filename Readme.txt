【1.はじめに】

このアプリケーションはファイアーエムブレムifで登場するユニットの成長率を自動計算するツールです。

とりあえず簡易的に利用できるツールとしてご利用ください。

※現時点では情報が不足しているため、表示される内容に誤りがある可能性があります。

また、ユニットとクラスであり得ない組み合わせが表示されてしまいますが、
まだ誰が何のクラスになれるのかという情報が整理されておりません。
情報が集まり次第、修正をしていく方針です。

今後、DLC配信が予定されている第３ルートにも対応する予定です。



【2.使用方法】

■　Windowsの場合
FEif-Calculator.exeを起動してください。

■　Macの場合
FEif-Calculatorを起動してください。
